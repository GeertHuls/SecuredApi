OAuth2 demo

This site is a proof of concept for claims based authentication using OAuth2's Implicit grant flow.

The id server is built out of IdentityServer3 granting access to the resource server on behalf of the client's user. The client itself (this page) is a javascript spa app.

Details on IdentityServer3 visit the docs.
